# DinoFiles • Full Beta (static)

To jest wersja demo działająca w pełni w przeglądarce (bez serwera).

## Funkcje
- Miniatury (JPG/PNG oraz 1. strona PDF przez PDF.js)
- Zmiana kolejności plików (drag & drop)
- Podgląd (modal z lupą)
- JPG/PNG → PDF A4 (dopasowanie jak skan, białe tło)
- Scalanie PDF wg kolejności

## Deploy (Vercel)
1. Wrzuć `index.html` i `vercel.json` do repo na GitHub.
2. Połącz repo z Vercel → Deploy.
3. Podłącz domenę i gotowe.
